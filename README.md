vimrc
=====

my vim configuration
### How to install

    git clone https://github.com/deju/vimrc.git
    cp vimrc/vimrc ~/.vimrc
    git clone https://github.com/deju/vundle.git ~/.vim/bundle/vundle

    mkdir -p ~/.vim/syntax
    cp -rf ~/vimrc/syntax/* ~/.vim/syntax

open VIM, then run

    :BundleInstall
